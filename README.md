#  Calculator - React Native

Mobil dasturlash fanidan topshiriq uchun yaratilgan kalkulyator ilovasi.

##  Ilova ko'rinishi

Yashil rangdagi zamonaviy kalkulyator:
- Oq display qora chegarali
- Yashil tugmalar
- Barcha asosiy arifmetik amallar

##  Xususiyatlari

| Tugma | Vazifasi |
|-------|----------|
| `0-9` | Raqamlar |
| `+` | Qo'shish |
| `-` | Ayirish |
| `*` | Ko'paytirish |
| `/` | Bo'lish |
| `.` | O'nlik kasr |
| `=` | Natija hisoblash |
| `c` | Tozalash |

##  APK ni yuklab olish va o'rnatish

### 1-qadam: APK faylini yuklab olish

 [APK YUKLAB OLISH](https://expo.dev/accounts/michaelmike/projects/Calculator/builds/2b24cb0d-59ff-416c-a21d-c2fb6ca11c05)

Yoki QR kodni skanerlang:

### 2-qadam: Telefonda o'rnatish

1. **Yuklab olingan APK faylini toping** (odatda `Downloads` papkasida)
2. **APK faylini bosing**
3. **"Noma'lum manbalardan o'rnatish"** so'ralsa:
   - `Sozlamalar`  `Xavfsizlik`  `Noma'lum manbalar` ni yoqing
   - Yoki "Ruxsat berish" tugmasini bosing
4. **"O'rnatish" (Install)** tugmasini bosing
5. **"Ochish" (Open)** tugmasini bosing

### 3-qadam: Ilovadan foydalanish

1. Ilovani oching
2. Raqamlarni kiriting
3. Amal tugmasini bosing (+, -, *, /)
4. Ikkinchi raqamni kiriting
5. `=` tugmasini bosing - natija chiqadi!
6. `c` tugmasi bilan tozalash mumkin

##  Texnologiyalar

- **React Native** - mobil ilova framework
- **Expo** - build va deploy uchun
- **JavaScript** - dasturlash tili

##  Loyihani ishga tushirish (Developerlar uchun)

### Talablar
- Node.js (v18+)
- npm yoki yarn
- Expo CLI

### O'rnatish

```bash
# Reponi klonlash
git clone https://github.com/Muhiddinov20/Calculator.git

# Papkaga kirish
cd Calculator

# Paketlarni o'rnatish
npm install

# Ilovani ishga tushirish
npx expo start
```

### Telefondan ko'rish

1. **Expo Go** ilovasini yuklab oling:
   - [Android](https://play.google.com/store/apps/details?id=host.exp.exponent)
   - [iOS](https://apps.apple.com/app/expo-go/id982107779)

2. Terminal'da chiqqan QR kodni skanerlang

3. Ilova telefoningizda ochiladi!
